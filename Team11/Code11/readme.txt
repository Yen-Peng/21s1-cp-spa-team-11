How to compile and run autotester on CLion:
- Open CMake project in CLion by opening the folder with CMakeLists.txt, folder name is Code11
- Name of solution executable : autotester
- Path of solution : Team11/Code11/cmake-build-release/src/autotester/autotester
- Build project on CLion
- Run project on CLion
- CMake version 3.21.1
- Configuration : release
